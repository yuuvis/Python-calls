## Request URL
https://api.yuuvis.io/dms/objects/{objectId}/history

## Request headers
| Header                    | Type   | Meaning                                                                                             |
|---------------------------|--------|-----------------------------------------------------------------------------------------------------|
| Ocp-Apim-Subscription-Key | String | Subscription key which provides access to the API. Found on the Subscriptions page of your account. |
