## Request URL

https://api.yuuvis.io/dms/objects/{objectId}/versions/{versionNr}/contents/renditions/pdf

## Request headers

```
Ocp-Apim-Subscription-Key           string          Subscription key which provides access to this API. Found in your Profile.

```
